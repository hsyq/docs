# PM2 Docs 中文版

PM2是一个守护进程管理器，它将帮助您全天候管理和保持应用程序在线。

官网：https://pm2.keymetrics.io/

Github：https://github.com/Unitech/pm2

# 快速开始

# 安装

可以使用npm和yarn 安装。

npm i -g pm2

# 启动应用

使用pm2快速启动一个Node.js程序：

pm2 start app.js

此时，你的应用将以进程守护的方式在后台运行。

# 常用命令

pm2 list 
pm2 restart
pm2 reload
pm2 stop
pm2 delete
pm2 monit
pmn2 logs
